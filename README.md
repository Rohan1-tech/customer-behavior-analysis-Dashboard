# customer-behavior-analysis-Dashboard
Developed an end-to-end data analysis project to understand customer shopping behavior and improve business decision-making.

Performed data cleaning and feature engineering using Python, conducted business analysis using SQL (PostgreSQL), and built an interactive dashboard in Power BI.

Analyzed customer segments, subscription behavior, and product performance to identify key revenue drivers.

Provided actionable recommendations such as improving customer retention through loyalty programs, targeting non-subscribers, and optimizing discount strategies.
